Đặt các file âm thanh của bạn trong thư mục này, ví dụ:
- c.mp3: giọng NỮ (cô giáo) đọc chữ "Cờ" hoặc "C"
- success.mp3: âm thanh khen thưởng (ting / bravo / vỗ tay nhẹ)
- error.mp3: âm thanh báo chưa đúng (nhẹ nhàng, không gây giật mình)

Tên file phải trùng với nguồn trong index.html:
  <audio id="letterAudio">  -> sounds/c.mp3
  <audio id="successAudio"> -> sounds/success.mp3
  <audio id="errorAudio">   -> sounds/error.mp3
